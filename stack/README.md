# Stack

[STack](https://www.wikiwand.com/en/Stack_(abstract_data_type))

- LIFO (last in first out)
- time complexity: worst-case time complexity is constant: O(1)
- methods time complexity:
  - push: Constant — O(1)
  - pop: Constant — O(1)
  - peek: Constant — O(1)
  - empty: Constant — O(1)
  - size: Constant — O(1)
  - swap: Constant — O(1)
